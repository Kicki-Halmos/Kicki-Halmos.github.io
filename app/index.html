Utbildningen

Utbildningen Webbutvecklare fullstack open source ger studenten en bred programmeringskompetens inom både backend och frontend.

Efter avslutad utbildning ska studenten ha kunskaper om/i:
1.1 Avancerade dynamiska och databasdrivna webblösningar med hänsyn till prestanda, stora datamängder och olika
1.2 Utveckling av webbapplikationer i bl a JavaScript (frontend och backend) och PHP för komplexa miljöer med hänsyn till prestanda och större datamängder
1.3 Aktuella utvecklingsspråk och programmeringsmetoder inom frontend så som JavaScript, HTML och CSS
1.4 JavaScript-ramverk och bibliotek och hur de kan användas
1.5 Kommunikation med relations- och dokumentdatabaser som MySQL och MongoDB
1.6 Hur automatiserade tester kan användas för att förebygga fel
1.7 Aktuella CMS-plattformar, hur de är uppbyggda, fungerar och relaterar till varandra samt fördjupad kunskap om en plattform
1.8 Vilka förutsättningar som gäller för molnbaserade lösningar
1.9 Protokollet REST och hur det används
1.10 Vilka moment som krävs i alla led när ett projekt sätts upp inklusive versionshantering, konfigurering, testning, kvalitetssäkring och produktionssättning
1.11 Arbetsprocessen vid systemutveckling utifrån agil projektmetodik

Efter avslutad utbildning ska den studerande ha färdigheter i att:
2.1 Utveckla interaktiva webbapplikationer i bl a NodeJS, PHP och JavaScript för komplexa miljöer med hänsyn till prestanda och stora datamängder
2.2 Utveckla komplexa applikationer i JavaScript, olika JavaScript-bibliotek samt med HTML och CSS
2.3 Genomföra problemlösning och felsökning i kod
2.4 Programmera mot databaser som MySQL och MongoDB
2.5 Konfigurera och skapa avancerade tillämpningar med ett CMS
2.6 Utveckla och produktionssätta molnbaserade lösningar
2.7 Skapa diagram för kravmodellering, analys och design av datasystem
2.8 Göra anrop mot REST-tjänster
2.9 Kunna genomföra de moment som krävs när ett projekt sätts upp inklusive versionshantering, konfigurering, testning, kvalitetssäkring och produktionssättning
2.10 Ta fram kravspecifikationer och skapa programmeringslösningar utifrån dessa
2.11 Arbeta med projektbaserad agil arbetsmetodik i webbproduktion
2.12 Kommunicera åtaganden och lösningar inom programmering på engelska

Efter avslutad utbildning ska den studerande ha kompetenser för att:
3.1 Efter en kravspecifikation, och enligt branschstandard, utveckla optimerade webbapplikationer med t ex JavaScript (frontend och backend), PHP, något JavaScript-bibliotek och ramverk, HTML och CSS
3.2 Utifrån en kravbild kunna välja optimal typ av lösning för en tjänst/applikation också med hänsyn till exempelvis prestanda
3.3 Efter kravspecifikation och utifrån givna förutsättningar kunna utforma en CMS-lösning
3.4 Samarbeta i agila projekt och bidra med specialiserade kunskaper inom webbutveckling samt medverka vid utveckling, planering, och genomförande av projekt
3.5 Ta ansvar för sin egen kompetensutveckling och följa med i bransch- och teknikutvecklingen på ett sådant sätt att det leder till vidare lärande

Frontend 1 - JavaScript

Syfte och mål: Syftet med kursen är att ge de studerande förståelse för grundläggande programmering och kompetensen att utveckla programmeringslösningar i JavaScript. De studerande ska även ges en förståelse för vad yrkesrollen innebär, relationen mellan frontend, backend och fullstack samt en förståelse för begreppet open source.

Målet är att de studerande ska kunna göra enklare applikationer i JavaScript och kunna skriva, testa och felsöka kod. De studerande lär sig hur språket skrivs, hur det exekveras samt om språkets olika möjligheter och begränsningar. De lär sig olika datatyper i språket som t.ex. string, number, array, object, m.m. Kursen behandlar även den miljö som JavaScript exekveras i och dess koppling till DOM och HTML. I kursen introduceras grundläggande krav- och versionshantering. Förkunskaper från denna kurs är en förutsättning för att gå ut på LIA.

De studerande ska ha kunskaper i/om:
• Fördjupad teoretisk förståelse för programmering i JavaScript och dess koppling till DOM och HTML
• Funktioner, styrsatser, variabler och datatyper
• Verktyg, effektivisering, felsökning
• Olika rekommenderade sätt att namnge variabler och metoder
• Hur interaktiva webbsidor kan skapas med hjälp av JavaScript
• Hur versionshantering används i ett programmeringsprojekt
• Hur arbetet med grundläggande kravhantering fungerar
• Vad det innebär att arbeta med open source-tekniker
• Relationen mellan frontend och backend
• Hur yrkesrollen som webbutvecklare fullstack ser ut

De studerande ska ha färdigheter i att: • Självständigt programmera och skapa enklare applikationer med hjälp av JavaScript
• Strukturera sin kod enligt kodkonventioner för JavaScript och göra den lättläst för andra programmerare
• Felsöka och testa kod
• Manipulera DOM med JavaScript
• Använda grundläggande versionshantering för att checka in och checka in kod samt hantera vanliga situationer som kan uppkomma, exempelvis merge-konflikter
• Beskriva - oberoende av programkoden - den uppgift ett program skall lösa och de förutsättningar som krävs för att det skall kunna arbeta

Frontend 2 - HTML & CSS

Syfte och mål: Syftet med kursen är att ge de studerande kunskaper och färdigheter om HTML och CSS för att skapa en interaktiv upplevelse med ett dynamiskt innehåll på en webbsida. De studerande lär sig att anpassa webbsidor efter olika förutsättningar genom responsiv webbdesign.

Målet är att de studerande genom praktiska övningar ska lära sig koda i HTML och CSS och kunna konvertera en designskiss till HTML och CSS. Utöver det lär de sig vikten av användbarhet och hur en applikation kan utvecklas på olika sätt med hänsyn till användbarhet och tillgänglighetsstandarder. Under kursen arbetar de studerande bl.a. med ett projekt där de skapar en interaktiv upplevelse med ett dynamiskt innehåll som ska kunna tillgängliggöras för externa användare. Kursen ger även en introduktion till agila arbetsmetoder för att de studerande ska ha förståelse för tidsestimering och projektmetodik som används i branschen. Förkunskaper från denna kurs är en förutsättning för att gå ut på LIA.

De studerande ska ha kunskaper i/om:
• Att skapa applikationer med HTML, CSS och JavaScript
• Olika layoutmodeller som används inom HTML och CSS utveckling för webbsidor
• SASS/LESS, hur det fungerar och hur teknikerna används för att generera CSS-kod
• Den miljö som JavaScript exekveras i och dess koppling till DOM och HTML
• Att skapa responsiva webblösningar som innehåller effektiv och optimerad kod som körs på klienten
• Vikten av användbarhet och hur en applikation kan utvecklas på olika sätt med hänsyn till användbarhet
• Olika tillgänglighetsstandarder och vilka regler som gäller för olika verksamheter
• Olika webbläsare och deras kompatibilitet
• Relevant projektmetodik med fokus på agil projektmetodik

De studerande ska ha färdigheter i att:
• Skapa applikationer med HTML, CSS och JavaScript
• Kunna använda SASS/LESS för att generera CSS-kod
• Skapa interaktiva webbsidor med hjälp av JavaScript
• Skapa responsiva webblösningar som innehåller effektiv och optimerad kod som körs på klienten
• Arbeta efter agil projektmodell.
